# Shoe Store - Интернет-магазин обуви

Современный интернет-магазин обуви, разработанный на React с использованием TypeScript и Redux Toolkit.

## Технологии

- **React 19** - библиотека для создания пользовательского интерфейса
- **TypeScript** - типизированный JavaScript
- **Redux Toolkit** - управление состоянием приложения
- **React Router** - маршрутизация
- **Vite** - сборщик и dev-сервер
- **Bootstrap 4** - CSS-фреймворк для стилизации

## Функциональность

### Основные возможности

- **Каталог товаров** с фильтрацией по категориям
- **Поиск товаров** по названию
- **Страница товара** с детальной информацией
- **Корзина покупок** с сохранением в localStorage
- **Оформление заказа** через API
- **Адаптивный дизайн** для мобильных устройств

### Управление состоянием

Приложение использует Redux Toolkit для централизованного управления состоянием:

- **cartSlice** - управление корзиной покупок
- **catalogSlice** - управление каталогом, категориями и поиском
- **topSalesSlice** - управление хитами продаж
- **productSlice** - управление страницей товара
- **uiSlice** - управление UI состоянием (поиск, меню)

## Установка и запуск

### Требования

- Node.js 20.19.0 или выше
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`

### Предпросмотр продакшен-сборки

```bash
npm run preview
```

### Проверка кода линтером

```bash
npm run lint
```

## Особенности реализации

### Корзина покупок

- Хранение данных в localStorage
- Одна позиция = товар + размер
- Фиксация цены при добавлении в корзину
- Адаптивный дизайн (таблица на десктопе, карточки на мобильных)

### API интеграция

Приложение работает с REST API для получения данных о товарах, категориях и оформления заказов.

### Адаптивность

Полностью адаптивный дизайн с поддержкой:

- Десктопных экранов
- Планшетов
- Мобильных устройств
